# Circling Landing Page

Static HTML/CSS site for GitHub Pages.

## Files
- `index.html` — page markup and expandable principle sections
- `styles.css` — all styling and background-image reference
- `assets/circling-background.png` — background image used by the page

## GitHub Pages
Upload these files to a GitHub repo, keeping the same folder structure. In repository settings, enable GitHub Pages from the main branch/root.
